《眼裡的美》

即使，
畫了少女般的粉嫩裝容，
依舊遮蓋不了她的高齡。
即時，
做了一次次的雷射除斑，
依舊改變不了年長的氣息。

高中生嘲諷四五十歲的女人，
畫了妖豔的妝，
只顯得她的假。

熟女看著畫著妝的高中生，
是嘆，
逝去的青春，
還是嘆，
年紀輕輕就在裝熟女？

原來美跟氣質，
相輔相成。
