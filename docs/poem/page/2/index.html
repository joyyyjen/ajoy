《這是什麼味道？》

2018.1.31

味

75點。星巴克。
叫了一個帕尼尼，一個三明治，一個楓糖拿鐵。
狼吞虎嚥的吃完，眼眶泛淚。
說一句話，體力都被抽走了。
免疫力歸零，吃進去的都出來了。

綠色的是沙拉，紅色的是蘋果。
側著頭思考的樣子，
心裏認定的是麥當勞炸雞。

《致秋冬》

2018.1.14

早，冬天

冬天的花草一副奄奄不息的樣子，
就算得了維納斯的喜悅。
親親他們的葉子，
顫抖害羞的葉子。
或許是鞠躬，
抖了抖結霜的露珠，
不敢抬頭看看面前的美人。
