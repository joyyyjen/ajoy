《這是什麼味道？》

2018.1.31

味

75點。星巴克。
叫了一個帕尼尼，一個三明治，一個楓糖拿鐵。
狼吞虎嚥的吃完，眼眶泛淚。
說一句話，體力都被抽走了。
免疫力歸零，吃進去的都出來了。

綠色的是沙拉，紅色的是蘋果。
側著頭思考的樣子，
心裏認定的是麥當勞炸雞。
